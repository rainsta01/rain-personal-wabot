# Rain Personal WhatsApp Bot

A simple unofficial WhatsApp bot built with Node.js using the Baileys library.  
This project is intended for learning, experimentation, and personal use.

---

## Features

- Basic WhatsApp bot functionality  
- Lightweight and easy to modify  
- Built using Baileys (WhatsApp Web API)  
- Uses Indonesian language for console logs  

---

## Installation

### Option 1: Clone Repository (Recommended)

```bash
git clone https://github.com/rainsta01/rain-personal-wabot
cd rain-personal-wabot
npm install
npm start
```

### Option 2: Download Latest Release (ZIP)

1. Go to the Releases page:  
   https://github.com/rainsta01/rain-personal-wabot/releases

2. Open the **latest release version**

3. Download the **rain-personal-wabot.zip** file from Assets

4. Extract the ZIP file on your computer

5. Password :
```bash
123
```

5. Open a terminal inside the extracted folder

6. Run:

```bash
npm install
npm start
```

---

## Usage

```bash
npm start
```

- A QR code will appear in the terminal  
- Scan it using WhatsApp (Linked Devices)  
- The bot will connect automatically  

---

## Troubleshooting

```bash
rm -rf node_modules
npm install
npm start
```

Or manually:
1. Delete `node_modules`  
2. Run `npm install`  
3. Run `npm start`
